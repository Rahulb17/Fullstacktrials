Project: Detailed Job Application Tracker (To-Do List)

Summary: A web application using HTML, CSS (Project 2 styling), and JavaScript for tracking job applications with multiple fields (Company, Role, Date, Status).

Implemented Features:
- Styled input form card (based on Project 2's centered design).
- JavaScript logic for adding, deleting, and archiving applications.
- Two distinct lists: Active and Archived.
- Custom color-coded status tags (Applied, Interviewing, Offer, Rejected).
- DOM manipulation for dynamic list updates.

Known Issues: Data persistence (local storage) is not implemented; data resets on refresh.
Browser Tested: Chrome, Firefox
