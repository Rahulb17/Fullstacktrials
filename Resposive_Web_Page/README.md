Project: Responsive One-Page Website

Summary: A single-page website featuring a sticky navigation bar, smooth scrolling, and a CSS-only hamburger menu for mobile responsiveness. Uses semantic HTML and media queries.

Implemented Features:
- Semantic HTML tags (<header>, <nav>, <section>, etc.).
- Sticky navigation bar (position: sticky).
- Smooth scrolling between 5 required sections (#home, #about, etc.).
- Responsive design using media queries (< 768px).
- CSS "Checkbox Hack" for the hamburger menu toggle.

Known Issues: None.
Browser Tested: Chrome, Safari, Mobile Chrome (Emulation)
