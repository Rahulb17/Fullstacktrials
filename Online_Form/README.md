Project: Centered Login Form

Summary: A responsive login form created using HTML and CSS, demonstrating correct application of the CSS Box Model (using box-sizing: border-box) and vertical/horizontal centering via Flexbox on the body element.

Implemented Features:
- Card with fixed width (350px) including padding and border.
- Card centered vertically and horizontally in the viewport.
- Full-width input fields and submit button.

Known Issues: None.
Browser Tested: Chrome, Firefox
